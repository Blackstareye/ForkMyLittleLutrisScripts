# Jazz Jackrabbit 2 - GOG - DxWnd - The Secret Files
### Description:
"Jazz Jackrabbit 2: The Secret Files" provided by GOG using DxWnd open source project
### Technical notes:
- will install Jazz Jackrabbit 2: The Secret Files  from GOG.
