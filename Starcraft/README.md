# Starcraft - CD + cnc-ddraw
### Description:
"Starcraft" CD edtion using cnc-draw
### Technical notes:
- Do not launch Starcraft during install process.
- Game will be updated to 1.16.1
- You will need Starcraft CD only one time during install.

# Starcraft - Battle.net launcher
### Description:
Battle.net launcher for Starcraft
### Technical notes:
- Before to install this script, you have to install "Battle.net - Standard" script installer
- You have battle.net installed and want an icon to launch directly "StarCraft - Battle.net edition" from Lutris interface, this script is for you
