# MLLS - Virtua Fighter PC - CD + DxWnd

### Description:
"Virtua Fighter PC" CD edition using DxWnd
### Technical notes:
- Controls:
[Alt + F4]	Quit
[F5]	Key configuration (Joystick, X360 compatible)
[F6]	Screen setting
[F7]	Screen brightness, camera, and volume settings
